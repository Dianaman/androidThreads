En Java cualquier instancia de la clase Thread corre en otro plano
Debemos llamar desde nuestro hilo principal al método de instancia start del objeto que extiende de Thread

public class MiHilo extends Thread {}

MiHilo hilo = new MiHilo();
hilo.start();

luego debemos sobreescribir en MiHilo el método run

@Override
public void run(){
}
